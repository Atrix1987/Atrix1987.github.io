Android Google Cloud Messaging Client Library

GCMLib is a library which manages GCM tokens, messages and delegates the state change or messages to appropriate callbacks. It comes in handy when you have multiple Push notification providers (for whatsoever reason it is) and seamlessly provides data to all the callbacks.

Library Health

Adding the library to your android project

                    
repositories {
    maven {
        url 'https://dl.bintray.com/atrix1987/maven/'
    }
}

dependencies {
    compile 'com.nandi:gcm-lib:1.0.1'
}

Integrating the library is just 3 lines of code


GcmHelper.getInstance()
    .setAuthorizedEntity("889908101771")
    .addRegistrationCallback(getApplicationContext(), new CustomRegistrationListener(), true)
    .addOnMessageReceivedCallback(new CustomMessageReceivedListener())
    .init(getApplication());
